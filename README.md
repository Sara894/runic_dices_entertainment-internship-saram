# Game Dev Internship Trial – Sara Mitrevska

This project is part of my internship trial and demonstrates my skills in programming and using Unity Engine

All tasks were developed using Unity and C#.

- Player sprites: "Isometric Orc Warrior Male" by Engvee ( https://engvee.itch.io/isometric-orc-warrior-male )
- Enemy sprites: "Isometric Halberd Warrior" by Engvee (https://engvee.itch.io/animated-isometric-halberd-warrior )
- UI sprites: "UI User Interface - medieval" by ToffeCraft (https://toffeecraft.itch.io/ui-user-interface-pack-medieval)
- Inventory sprites: "Forgotten Vials – Vol. 1 (13 FREE POTIONS)" by djinnbestiary ( https://djinnbestiary.itch.io/forgotten-vials-vol-1-13-free-potions )
